## Industry Overview
Developers and owner-builders manage capital, risk, and timelines across multi-phase projects.
They care more about cash flow, lender requirements, and phase gating than daily field tasks.

## Typical Workflow
- Acquisition
- Feasibility & Pro Forma
- Design & Entitlements
- Construction
- Lease/Sale
- Operations / Exit

## Key Tasks by Phase
Acquisition: site evaluation, zoning review
Pro Forma: budget, ROI, financing assumptions
Design/Entitlement: approvals, variances
Construction: draw schedules, inspections
Closeout: CO, lender signoff
Operations: warranty and asset management

## Must-Have Fields
- Pro forma budget
- Funding sources (equity, debt)
- Draw schedule
- Lender inspections
- Phase approvals
- Exit strategy

## Critical Screens
- Financial Dashboard
- Phase Gate Tracker
- Draw Request Manager
- Risk Register
- Portfolio View
