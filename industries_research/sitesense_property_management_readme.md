# Industry ReadMe – Property & Facilities Management (Sitesense)
_Generated: 2025-12-19_

This document defines workflows, data requirements, and system capabilities for Property Management and Facilities Management organizations.

## 1) Industry Overview
Property managers run recurring service operations across multiple properties. Speed, documentation, and cost control matter more than project delivery.

## 2) Core Workflow
Request Intake → Triage → Vendor Assignment → Execution → Verification → Billing → Asset History

## 3) Core Modules
- Property & Unit Management
- Tenant / Occupant Portal
- Work Order Management
- Vendor Management & Rate Cards
- SLA & Priority Tracking
- Preventative Maintenance
- Billing & Owner Reporting

## 4) Workflow by Phase
### Request Intake
- Tenant or occupant submits request
- Categorize priority and trade

### Triage
- Approve work and budget
- Determine NTE

### Vendor Assignment
- Dispatch approved vendor
- Coordinate access

### Execution
- Vendor completes work
- Capture photos and notes

### Verification
- Manager review
- Tenant confirmation

### Billing
- Vendor invoice
- Owner allocation

### Asset History
- Update asset records
- Schedule preventative maintenance

## 5) Must-Have Fields
- Property / Unit
- Work Order Priority & SLA
- Vendor Rate Card
- Preventative Schedule
- Invoice Allocation

## 6) Critical Screens
- Work Order Queue
- SLA Timers
- Vendor Dispatch Board
- Asset History Timeline
- Owner Reports
